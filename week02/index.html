Chapter 2 - Programming Basics

  • Variables declared with let can be redefined, while const variables cannot. let and const are both block-scoped.
  • var is the old way to declare variables. It's not recommended.
  • Template literals are strings surrounded by backticks ` that allow you to embed expressions in them. This is called string interpolation. Expressions are wrapped in ${...}. 
      const name = "Eddie"
  console.log(`My name is ${name}!`)
  // Outputs: My name is Eddie!
  • The Number() and String() functions can be used to convert between numbers and strings.
  • As an alternative to Number(), you can use parseInt() which will parse the first number it finds in a string. You can also specify the base to convert the number to.
  • parseFloat() is similar to parseInt(). However, it doesn't round the number, it truncates it.

Chapter 3 - Array, Logic, and Loops

  • You can use array destructuring to swap two variables. 
      [x,y] = [y,x]
  • Several useful array methods include: 
      Array.pop()     // Removes element from the end of the array
  Array.push()    // Adds element to the end of the array
  Array.shift()   // Removes element from the start of the array
  Array.push()    // Adds element to the start of the array
  Array.slice()   // Returns a subarray (non-destructive)
  Array.splice()  // Insert/remove values from an array (destructive)

Chapter 4 - Functions

  • The rest operator (...) is used to collect arguments passed to a function and store them in an array.
  • Functions and variables declared with var are hoisted to the top of their scope, meaning they can be used before they are declared. ES6 arrow functions and block-scoped variables are not hoisted.
  • Array iterators, such as the following, can be used for various array methods, and can even be chained together. 
      Array.foreach()  // Uses a callback function on each element in the array
  Array.map()      // Similar to foreach, maps each value to a new value
  Array.reduce()   // Reduces items in an array down to a single value
  Array.filter()   // Filters out items from an array and returns a new array
