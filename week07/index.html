Chapter 11 - Further Functions

  • The call method can be used by a function to specify a value for this, as well as take in a set of parameters. The apply method is similar, but takes in an array of parameters.
  • Functions are objects, and can have properties defined on them. This can be useful when implementing memoization (caching of results).
  • Immediately Invoked Function Expressions (IIFE) are a way to create a function that is immediately executed by wrapping the function declaration in parentheses and then invoking it.
  • IIFE's can be used to create temporary variables that are not accessible outside of the function or create a block where strict mode can be used without affecting existing code.
  • Init-time branching allows you to redefine the behavior of a function based on a condition, such as whether or not the browser supports a certain API.
  • Promises allow you to wait for a value that will be available later. Promises can be chained together using then to specify what should happen when the promise is resolved or rejected.
  • async and await allow you to wait for a promise to resolve before continuing.
  • Closures are created by returning a function from another function, where the inner function has access to the variables in the outer function. Even after the outer function has returned, the inner function still has access to the variables in the outer function.
  • Generator functions are created by using function* and the yield keyword. After calling the generator function, the next method can be called to get the next value.
  • Generators work as iterators, and can be used in for-of loops.
  • Curried functions are higher-order functions that take in multiple arguments. If all arguments are provided, the function will behave like normal. If only some of the arguments are provided, the function will return a new function that takes in the remaining arguments.

Chapter 13 - AJAX

  • AJAX was created to allow webpages to send and receive data from a server without having to reload the page. More recently, the fetch API was introduced to perform similar requests. The fetch API is promise-based.
