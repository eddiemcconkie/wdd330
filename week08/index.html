Chapter 8 - Transforms and Transitions

  • translate(), rotate(), scale(), and skew() can be used to transform elements.
  • transform-origin can be used to change the origin of the transformations applied to an element. By default, the origin is the center of the element.
  • transition can be used to animate the properties of an element, such as highlighting a button when you hover over it. There are various transition properties to affect the duration, timing, and delay of the animation.
  • animation works with @keyframes to create animations with finer control.

Chapter 12 - Canvas, SVG, and Drag and Drop

  • The canvas API can be used to draw shapes and images on the page. Most browsers support 2D drawings, although some support 3D.
  • SVG's are scalable vector graphics. They are defined in XML and can be included in the DOM. Unlike bitmap images, SVG's can be scaled, rotated, and transformed without any loss of quality.
  • DOM elements can be made draggable allowing them to be dragged and dropped around the page.
