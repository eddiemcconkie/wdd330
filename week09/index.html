Chapter 9 - The Window Object

  • The alert, confirm, and prompt methods are all part of the window object, and will stop any code from running until the user has responded.
  • The window.location has various properties related to the current page's URL.
  • The window.history object keeps track of the session history of the page.
  • The setTimeOut and setInterval methods are used to run functions after a certain amount of time.
  • The window has the requestAnimationFrame function to help with animating, but it's usually better to animate with CSS.

Chapter 14 - HTML5 APIs

  • data- attributes allow you to store data in an HTML element that can be accessed through JavaScript.
  • Data can be stored in the browser using the Web Storage API.
  • The Geolocation API gives you access to the user's location.
  • Workers allow you to run concurrent JavaScript in the background. You can send messages between the main thread and the worker thread.
  • Service Workers can be used to cache resources and use them offline, or can be used for handling push notifications and background sync.
  • Websockets allow immediate communication between the client and server.
  • The Notification API allows you to send push notifications to the user.
  • The Multimedia API allows you to play audio and video files. There are various event listeners that can be used, and the loadmetadata event should be used to get information about the media once it's loaded.
  • Polyfills help make new features available in older browsers.
